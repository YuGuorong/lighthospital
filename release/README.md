# 轻量级诊所管理系统 - Windows 版本

## 系统简介

这是一个专为小型诊所设计的轻量级管理系统，基于 Go 语言和 Gin 框架开发，使用 SQLite 数据库，支持局域网部署。

## 功能特性

- ✅ **用户认证管理** - 支持管理员和医生角色
- ✅ **患者信息管理** - 完整的患者档案管理
- ✅ **药品库存管理** - 药品信息、库存、预警
- ✅ **处方管理** - 电子处方开具、查询、打印
- ✅ **预约管理** - 患者预约安排和状态跟踪
- ✅ **操作日志** - 完整的操作记录和审计
- ✅ **局域网访问** - 支持多台设备同时访问
- ✅ **数据导出** - 支持处方和预约单打印

## 系统要求

- Windows 7/8/10/11 (64位)
- 无需安装 Go 语言环境
- 无需安装数据库
- 支持局域网部署

## 快速开始

### 1. 解压文件
将下载的压缩包解压到任意目录，例如：`C:\lighthospital`

### 2. 启动系统
双击运行 `start.bat` 文件，或在命令行中执行：
```cmd
start.bat
```

### 3. 访问系统
- 本地访问：http://localhost:8080
- 局域网访问：http://[本机IP]:8080

### 4. 登录系统
- **管理员账号**：admin / admin123
- **医生账号**：doctor / doctor123

## 网络配置

### 局域网访问
1. 确保防火墙允许 8080 端口访问
2. 在 Windows 防火墙中添加例外规则
3. 其他设备通过 `http://[本机IP]:8080` 访问

### 端口配置
默认端口：8080
如需修改端口，请编辑 `lighthospital.exe` 的启动参数

## 数据安全

- 所有数据存储在本地 SQLite 数据库中
- 支持定期数据备份
- 操作日志记录所有重要操作
- 用户密码采用 bcrypt 加密存储

## 故障排除

### 常见问题

1. **无法启动程序**
   - 检查是否被杀毒软件拦截
   - 确保有足够的磁盘空间
   - 检查端口 8080 是否被占用

2. **无法访问网页**
   - 检查防火墙设置
   - 确认程序正在运行
   - 检查浏览器是否支持

3. **数据库错误**
   - 删除 clinic.db 文件重新初始化
   - 检查磁盘权限
   - 确保磁盘空间充足

### 日志查看
程序运行日志会显示在控制台窗口中，包括：
- 数据库初始化信息
- 用户登录记录
- 错误信息

## 技术支持
